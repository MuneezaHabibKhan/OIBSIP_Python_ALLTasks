# Chat Application (Python)

This is a simple client-server based chat application developed using Python sockets.  
It allows multiple clients to communicate with the server in real time.

## Features
- Client–Server architecture
- Real-time message exchange
- Implemented using Python socket programming
- Command-line based interface

## Technologies Used
- Python
- Socket Programming
- TCP/IP

## Files Description
- `server.py` – Starts the server and handles multiple client connections
- `client.py` – Connects to the server and sends/receives messages

## How to Run the Application

### Step 1: Start the Server
Open a terminal and run:
```bash
python server.py
